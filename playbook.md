## Getting Started

- Clone this repo: `git clone https://github.com/fengweit/research`
- Review existing notes before starting a new topic
- Create a new folder per research topic

## Research Process

1. **Define the question** — What are you trying to learn or solve?
2. **Gather sources** — Articles, papers, docs, experiments
3. **Summarize findings** — Keep notes in this repo
4. **Draw conclusions** — What did you learn? What's next?

## Documentation Standards

- Use Markdown for all notes
- Name files clearly: `topic-subtopic.md`
- Date entries where relevant: `YYYY-MM-DD`
- Link to sources
